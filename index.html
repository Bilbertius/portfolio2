Try-lingual

Try-lingual is a language learning app that allows users to learn foreign vocabulary. Try-lingual uses the spaced repetition learning technique to help users memorize the words for ever increasing lengths of time.
Currently, the app only comes in German. The front-end was built with React while the on the backend an express API was built to connect the front-end with our PostgreSQL database.

Indecisio

This app was built with react, express, and postgresql with the goal in mind of providing people with a way to find inspiration when at a loss concerning how to spend their free time. Users can submit activities that they think others might enjoy, and thus the pool of possible activities can grow organically over time. When submitting an activity the user must pick a category to such as learning, entertainment, fitness, etc. that the activity falls under. The user generating an activity can then narrow down the possible activity suggestions by selecting a category. The user can accept or reject as many activities as they wish. The app also keeps track of the most popular activites, both of the user and from all users.
